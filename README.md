# Getting Started with Create React App

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Frontend Configuration (React)

1. Navigate to the frontend project folder.
2. Install Dependencies:  
   Run the following command to download all necessary React packages:
   _yarn install_
3. Link the API:
   Open _src/services/apiService.ts_ and update the _API_BASE_URL_ variable with the backend URL you copied earlier.
   _export const API_BASE_URL = "https://localhost:7001";_
4. Run the App:
   _yarn start_
